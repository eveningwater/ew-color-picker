# ewColorPicker

一个使用原生JavaScript编写的灵活性高扩展强的颜色选择器。

  • 介绍文章---思否 (opens new window)
  • 介绍文章---掘金 (opens new window)

# 使用场景

这个颜色选择器适用于中小型项目，例如主题的切换。不同于组件库中的颜色选择器组件，它的配置自主化，根据用户的需求来自定义。

# 特点

html5的原生颜色选择器样式不好看，而组件库的颜色选择器不够灵活多变，这样一来，就有了这个颜色选择器的诞生。
